Quickstart installation guide

For Python 3. If you do not have Python 3 installed on your machine, get the latest version here.

More detailed installation documents are available here.

1. Install datakit-core

This is the basic command line interface.

terminal

  • pip install datakit-core

2. Install datakit-project

This is the project structure plugin.

terminal

  • pip install datakit-project

3. Grab a template

For project structures, any valid Cookiecutter will work.

terminal

  • datakit project create -t https://github.com/associatedpress/cookiecutter-r-project.git

4. Get to work

terminal

  • datakit project create

5. Adapt to your workflow

Grab other plugins or develop your own!

  • datakit-data: Amazon S3 sync

  • (coming soon!) datakit-dworld: Data.world sync

  • (coming soon!) datakit-github: Github integration
